Jukebox
Overview Details API Demos

Sophisticated Audio Playback for the web

Jukebox is a component to handle the playback of music and sound effects across multiple web browsers and operating systems.
It is designed for ultra-low performance environments, such as mobiles - and is even able to deliver Android 1.6+ via Flash fallback.

Download

You can also clone the project with Git by running: 

$ git clone git://github.com/zynga/jukebox

Get the source code on GitHub : zynga/jukebox

Install

Installation is pretty simple. Just copy the files located in the src/ folder to your project and include them in the following order:

  • src/Player.js
  • src/Manager.js

Flash Fallback

The (alternative) Flash fallback requires the compiled swf from src/swf/FlashMediaElement.as, a compiled version is located in the demo/media/swf folder.

Make sure to pass through the correct URL to the swf file.
See jukebox.Player Settings for more details.
